
# Mini-process-priotity-controller

### Installation:

**Clone repo**
```
git clone https://github.com/callmemikhail/mpc.git
cd mpc/mini-prc-ctl
```
**Execute permissions**
```
chmod u+x mini-process-controller.sh
```

### Usage:
```
./mini-process-controller.sh -n librewolf -p -20 -t 10
```
* flag [-t] -- time period
* flag [-n] -- process name
* flag [-p] -- process priotity
